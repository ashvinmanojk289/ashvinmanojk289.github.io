Case Study: PDF Query Application

Problem: Extracting specific information from large PDF documents is time-consuming and inefficient.

Solution: I developed a scalable, voice-enabled system using NLP models to transform static PDFs into conversational hubs. Users can ask questions in natural language (or voice) and receive instant, contextually accurate answers. This reduced information retrieval time by over 89%.

Tech Stack: Python, Streamlit, OpenAI API (for NLP), Google Speech Recognition, gTTS.

Case Study: Automatic Weed Detection and Spraying Robot

Problem: Traditional herbicide application in agriculture is often imprecise, leading to environmental damage and wasted resources.

Solution: I built an AI-powered autonomous robot that uses computer vision to differentiate between crops and weeds with 97% accuracy. It navigates fields using ROS and applies herbicide directly to weeds, minimizing environmental impact. The core of the system is a custom-trained hybrid EfficientNetV2-Transformer model for robust detection.

Tech Stack: Python, ROS, PyTorch, Transformers, Computer Vision (OpenCV), Embedded Systems.

Case Study: Multilingual News Audio Translator

Problem: Live news broadcasts are often inaccessible to audiences who speak different languages.

Solution: I developed a full-stack web application that translates spoken audio in near real-time. It uses a Wav2Vec 2.0 model for highly accurate speech recognition and a fine-tuned mBART model for fluent, context-aware translation between multiple languages.

Tech Stack: Python, Flask, Hugging Face (Transformers, Wav2Vec 2.0, mBART), PyTorch.
