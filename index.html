Portfolio

  • PDF Query Application

    GenAI / NLP

    Created a Python tool for querying and extracting information from PDF documents using natural language.

    GitHub

    Problem

    Manually searching large PDF documents for specific information is time-consuming and inefficient. Standard search (Ctrl+F) lacks contextual understanding and cannot answer natural language questions.

    Solution

    I built a Python tool that leverages NLP to parse and understand PDF content. It creates a searchable index, allowing users to ask questions in plain English (e.g., "What was the total revenue in 2024?") and receive precise answers. This automates information extraction and streamlines document-based workflows.

  • Hybrid EfficientNetV2-Transformer Model for Weed Detection

    Deep Learning / Computer Vision

    A research project comparing deep learning architectures for weed detection in agriculture.

    GitHub

    Problem

    Traditional agricultural methods often involve broad-spectrum herbicide application, which is costly and environmentally harmful. Identifying weeds accurately, especially those that closely resemble crops, is a significant computer vision challenge.

    Solution

    I researched, built, and benchmarked several deep learning models, including a novel hybrid architecture combining EfficientNetV2 (for strong feature extraction) and a Transformer (for contextual relationships). This hybrid model achieved 97% accuracy on large datasets, demonstrating a robust and efficient solution for precision agriculture.

  • Multilingual News Audio Translator (Flask UI)

    Full-stack / AI/ML

    A web app for translating news audio into multiple languages using speech-to-text and machine translation.

    GitHub

    Problem

    Access to news and information is often limited by language barriers. Audio content, like podcasts and news broadcasts, is particularly difficult to translate and consume in real-time.

    Solution

    I developed a Flask web application that integrates multiple AI models. It uses Wav2Vec 2.0 for speech recognition, an mBART model for machine translation, and a TTS engine to speak the translated text. This creates a seamless pipeline for users to consume audio news in various languages, improving accessibility.

  • Weed Detection Robot

    Robotics / Computer Vision

    An autonomous robotic system using computer vision for real-time weed detection and classification.

    GitHub

    Problem

    Manual weeding is labor-intensive, and non-selective spraying is inefficient. An autonomous solution needs to reliably distinguish between crops and weeds in real-time within a complex, unstructured field environment.

    Solution

    I engineered a complete robotic system integrating ROS (Robot Operating System) for navigation and control, with a computer vision module for perception. By applying image processing and a custom-trained ML model, the robot can identify and classify weeds in real-time, enabling future integration with precision spraying mechanisms.

  • Simple Google PageRank Algorithm

    Algorithm / Data Science

    An implementation of the PageRank algorithm in Python to simulate web link analysis and page ranking.

    GitHub

    Problem

    Understanding how search engines rank web pages is a fundamental concept in data science and web technology. The challenge is to mathematically model "importance" based on the web's link structure.

    Solution

    I implemented the Google PageRank algorithm from scratch in Python. This involves representing the web as a graph and iteratively calculating the "rank" of each page based on the quantity and quality of its inbound links. This project demonstrates a strong understanding of graph theory, linear algebra, and iterative algorithms.
